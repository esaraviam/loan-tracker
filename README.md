## Loan Tracked

A simple application to track loan

```bash
npm run dev
yarn dev
pnpm dev
bun dev
```

## Config

this app uses s3 to store images so you need to create a .env file with this
```
DATABASE_URL="file:./dev.db"
AWS_BUCKET_NAME 
AWS_REGION
AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY
AWS_BUCKET
AWS_BUCKET_FOLDER
AWS_BUCKET_URL
```
